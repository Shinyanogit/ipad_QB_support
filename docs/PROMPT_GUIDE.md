# Prompt Guide

## ChatGPTバー
- 既定のシステムプロンプトは「QB問題集の学習支援アシスタント」として回答する。
- 問題文/選択肢/進捗/タグ/画像をコンテキストに含める。
- 説明レベル（高校生/医低/医高）に応じたテンプレートを挿入。
- 共通プロンプトは常に付加される。
- 要点: 簡潔に、日本語で、不足情報は明示する。

## 例
- 質問: 「この問題のポイントは？」
- 質問: 「各選択肢の正誤理由を簡単に」
## テンプレート
- テンプレートは最大5件。ショートカットから送信可能。
- 「ヒント」テンプレートには400文字以内の制約文が含まれる。

## ヒント（解説コンテキスト）
- ヒント送信時のみ、解説本文を抽出して instructions に追記する。
- 抽出元は `#answerSection` または `div.contents-main`（未表示時は内部iframeから取得）。
- 追加される文面は以下の2行＋解説本文：
  - 「以下は解説本文の抽出です。この内容に完全準拠する必要はないが、矛盾しないヒントを提示してください。」
  - 「この解説本文を見ていることや内容を直接引用していることは出力に現さないでください。」
- 既存のユーザー考察がある場合は、ヒント文の末尾に `今の考察は以下です..."..."` を付与する。

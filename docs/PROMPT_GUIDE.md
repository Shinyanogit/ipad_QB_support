# Prompt Guide

## ChatGPTバー
- 既定のシステムプロンプトは「QB問題集の学習支援アシスタント」として回答する。
- 問題文/選択肢/進捗/タグ/画像をコンテキストに含める。
- 説明レベル（高校生/医低/医高）に応じたテンプレートを挿入。
- 共通プロンプトは常に付加される。
- 要点: 簡潔に、日本語で、不足情報は明示する。

## 例
- 質問: 「この問題のポイントは？」
- 質問: 「各選択肢の正誤理由を簡単に」
## テンプレート
- テンプレートは最大5件。ショートカットから送信可能。
- 「ヒント」テンプレートには400文字以内の制約文が含まれる。
